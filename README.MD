# DiskScope（磁盘空间透视眼）

一个轻量级的本地 Web 磁盘分析工具。只需在本地启动一个 Python 后端，浏览器中即可直观查看所有磁盘文件和文件夹的空间占用情况。左侧目录树、右侧交互式饼图，支持任意层级下钻，帮你快速找出“是谁吃掉了磁盘空间”。

![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 核心特性

- **双栏交互**：左侧目录树（懒加载，无卡顿）＋ 右侧可点击下钻的扇形图。
- **全盘扫描**：自动识别所有逻辑驱动器（C:/, D:/, ...），管理员权限绕过系统保护目录，遇到无权限静默跳过。
- **异步进度反馈**：后台线程扫描，前端实时展示进度百分比和当前遍历路径。
- **智能聚合**：大于 100 MB 的文件单独扇区显示，小文件聚合为“小文件（共XX MB）”，层级项超过 25 个时自动归入“其他”扇区，保证图表清晰。
- **安全只读**：绝不修改、删除任何文件，纯分析工具。

## 🚀 快速开始

### 环境要求
- Python 3.8 或更高版本
- pip 包管理工具
- **Windows：建议以管理员身份运行**（用于访问系统保护文件夹，否则部分目录可能因权限被跳过）

### 安装依赖

```bash
pip install -r requirements.txt
```
### 启动应用
在项目根目录执行：

```bash
python app.py
```

浏览器将自动打开 http://127.0.0.1:5000，看到界面后即可使用。

Linux / macOS：需以 sudo 或具有相应权限的用户运行，某些目录可能需要提升权限。


### 📖 使用说明
启动后，界面左侧显示“我的电脑”及所有盘符。

点击任一盘符（如 C:/），弹出扫描进度条（扫描耗时取决于磁盘文件数量，通常数秒到数十秒）。

扫描完成后，左侧目录树会展开盘符下一级目录，右侧扇形图显示该盘符直接子项的大小占比。

下钻操作：
```
点击左侧目录树中的文件夹 → 右侧扇形图更新为该文件夹的内容。

点击扇形图中的文件夹扇区 → 左侧树自动高亮并展开，图表下钻。

点击文件扇区或聚合项 → 弹窗显示详细信息（路径、精确大小等）。

顶部按钮可快速返回“我的电脑”或切换到父级目录。
```
### ⚠️ 注意事项
管理员权限：强烈建议以管理员身份运行，否则部分系统目录（如 System Volume Information）会被静默跳过，导致磁盘总大小统计略小于实际占用。

只读设计：工具仅遍历文件系统读取文件大小，不进行任何写操作，绝对安全。

网络通信：服务仅监听 127.0.0.1，不会暴露给局域网或公网，无需担心外部访问。

大磁盘处理：使用了 os.scandir 和聚合算法，即便百万级文件也能在可接受时间内完成，且前端图表不会因过多扇区而卡顿。

### ❓ 常见问题

Q：为什么某些文件夹扫描不到内容？
A：通常是权限不足。可尝试以管理员身份重新运行程序，程序会记录跳过路径，但不影响整体使用。

Q：能否导出分析报告？
A：当前版本未包含该功能，但可通过浏览器的开发者工具获取 JSON 数据自行处理，计划在后续版本中加入 CSV/HTML 导出。
